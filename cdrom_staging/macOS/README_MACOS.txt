SUNFLOWER AI - MACOS QUICK START
================================

To start Sunflower AI:
1. Open Finder
2. Click on "SUNFLOWER_AI" in the sidebar
3. Double-click "SunflowerAI.app"
4. If blocked, go to System Preferences > Security & Privacy and click "Open Anyway"

First Time Setup:
- Create a parent account with password
- Add profiles for each child
- The AI will adapt to each child's age automatically

Troubleshooting:
- macOS 10.14 (Mojave) or newer required
- Needs at least 4GB RAM
- May need to allow microphone/camera access

For help, see the full guide at README.pdf
