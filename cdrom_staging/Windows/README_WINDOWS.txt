SUNFLOWER AI - WINDOWS QUICK START
==================================

To start Sunflower AI:
1. The program should start automatically when you insert the USB
2. If not, double-click "SunflowerAI.exe"
3. Allow any security prompts from Windows

First Time Setup:
- Create a parent account with password
- Add profiles for each child
- The AI will adapt to each child's age automatically

Troubleshooting:
- If you see "Windows protected your PC", click "More info" then "Run anyway"
- Requires Windows 10 or newer
- Needs at least 4GB RAM

For help, see the full guide at README.pdf
